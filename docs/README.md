## Resumen de Funcionalidades

### Modulo de Ventas
- Busqueda de productos por codigo de barras o nombre
- Carrito de compras
- Facturacion electronica AFIP
- Envio de facturas por email
- Multiples formas de pago

### Modulo de Productos
- Alta, baja y modificacion de productos
- Gestion de stock por sucursal
- Transferencias entre sucursales
- Impresion de etiquetas con codigo de barras
- Imagenes de productos

### Modulo de Clientes
- Registro de clientes
- Datos fiscales para facturacion
- Historial de compras

### Modulo de Proveedores
- Gestion de proveedores
- Registro de pagos a proveedores

### Modulo de Reportes
- Ventas por periodo
- Stock por sucursal
- Cierre de caja
- Logs de auditoria

### Configuracion
- Perfil de empresa
- Sucursales y puntos de venta
- Usuarios y roles
- Integracion AFIP

## Roles de Usuario

| Rol | Acceso |
|-----|--------|
| Vendedor | Solo ventas |
| Vendedor+ | Ventas + clientes |
| Cargador | Carga de productos |
| Supervisor | Ventas, cargas, reportes |
| Administrador | Acceso completo |
